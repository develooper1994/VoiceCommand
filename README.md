# VoiceCommand

Türkçe sesli komut sistemi deneyidir. 

## Nasıl Çalışır?
dotnet ve [vosk](https://github.com/alphacep/vosk-api) ile yapılmıştır. Whisper daha iyi olsada çok fazla kaynak gerektirdiğinden vosk denemek istedim. 
[Vosk model listesindeki ](https://alphacephei.com/vosk/models) [Türkçe dil modelini](https://alphacephei.com/vosk/models/vosk-model-small-tr-0.3.zip) indirin. 
Programın çalıştığı dizinde model adında bir klasör açın ve indirilen zip dosyasının içindeki klasörü değil klasörün içini model dosyasına çıkarın.
